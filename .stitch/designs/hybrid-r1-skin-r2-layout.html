MODELS.

Browse the AI ecosystem from your terminal. A high-performance Rust interface for the structural exploration of large language models.

$ brew install models
Models 4,000+
Benchmarks ~400
Providers 22
Agents 11+
THE INTERFACE
[SYSTEM_CORE]: INITIALIZING TUI...

WHAT IT DOES

MODELS

Instant access to the complete metadata of the global AI ecosystem. Filter by parameter count, quantization, or license type directly from your shell.

BENCHMARKS

Raw performance data from MMLU to HumanEval. Compare latency and throughput across 22 major providers without leaving the terminal environment.

AGENTS

Track the active status of deployed autonomous agents. Monitor heartbeat, resource consumption, and task queues in a unified structural layout.

STATUS

Real-time endpoint health for OpenAI, Anthropic, Mistral, and local Ollama instances. Direct integration with systemd for background monitoring.

ALSO A CLI
$ models list --provider openai

Enumerate all available models from specific providers with JSON output support.

$ models benchmarks show gpt-4o

Retrieve specific evaluative scores and capability tiers for a target architecture.

$ agents status

Print a summary table of all active local and remote agent processes.
